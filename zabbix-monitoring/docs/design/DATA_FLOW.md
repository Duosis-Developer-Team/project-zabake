## ğŸ” DetaylÄ± Ä°ÅŸlem AdÄ±mlarÄ±

### Step 1: Data Collection

1. **Initialize Collector**
   - Load configuration
   - Connect to Zabbix API or Database
   - Authenticate

2. **Collect Hosts**
   - Fetch all hosts from Zabbix
   - Filter hosts if needed
   - Save to `hosts.json`

3. **Collect Templates**
   - Fetch all templates
   - Include template inheritance
   - Save to `templates.json`

4. **Collect Items**
   - Fetch items for each host
   - Include template items
   - Save to `items.json`

5. **Collect History**
   - Fetch history data for connectivity items
   - Filter by time range
   - Save to `history.json`

### Step 2: Template Analysis

1. **Load Templates**
   - Read `templates.json`
   - Build template hierarchy
   - Map templates to hosts

2. **Extract Items**
   - Get all items from templates
   - Map items to hosts
   - Create item-host relationships

### Step 3: Connectivity Item Detection

1. **Filter Items**
   - Apply connectivity patterns
   - Filter by item type
   - Filter by item key

2. **Create Connectivity Items**
   - Create ConnectivityItem objects
   - Link to hosts and templates
   - Add metadata

### Step 4: Data Analysis

1. **Load History Data**
   - Read `history.json`
   - Group by item ID
   - Sort by timestamp

2. **Analyze Each Item**
   - Check last data timestamp
   - Calculate data age
   - Determine active/inactive status

3. **Calculate Scores**
   - Calculate connectivity score per host
   - Identify issues
   - Generate statistics

### Step 5: Report Generation

1. **Load Analysis Results**
   - Read `analysis.json`
   - Prepare report data structure

2. **Generate Report**
   - Create report metadata
   - Add summary statistics
   - Add host details
   - Format output

3. **Save Reports**
   - Save in requested formats
   - Generate filenames with timestamps
   - Store in reports directory

- **Connectivity Detection**: 1-2 minutes
